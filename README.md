# baldwinit.tech (MVP)

Remote-first marketing + intake site for BaldwinIT, aligned to MSPGuild / OpsNerds styling.

## Run locally
docker compose up --build

Site will be available on the container network at port 80 (behind your reverse proxy).

## Env vars
- APP_NAME
- APP_URL
- CONTACT_EMAIL
- TICKET_URL (optional)
- BOOKING_URL (optional)
